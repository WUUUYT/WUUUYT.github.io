ISyE 6761 Lec 18 CTMC Definition

Posted on 2024-11-26 In Course Notes , ISyE 6761

Jump Matrix

Def
: Given a transition-rate matrix $Q$ on an alomost countable set $I$, the jump matrix $R=(r_{ij}: i,j\in I)$ of $Q$ is defined as follows. For any $i\neq j, i\in X$,

  1. $r_{ij}=0, r_{ii}=1$ if $q_i=0$.
  2. $r_{ij}=\frac{q_{ij}}{q_i}, r_{ii}=0$ if $q_i\neq0$.

    For each row with $q_i\neq0$, the diagonal is set to 0 and normalize the other entries. When $q_i=0$, the diagonal is set to 1.
    There is no self-loops, unless it’s on absorbing state

Def
: A minimal right continuous process $(X_t)_{t\geq0}$ on state space X is a continuous time markov chain with initial distribution $\nu$ and generator matrix $Q$ if

  1. its jump chain $(Y_n)_{n\geq0}$ is DTMC($\nu$, $R$), where $R$ is the jump matrix of $Q$.
  2. for each $n\geq1$, conditional on $Y_0,\ldots Y_{n-1}$, its holding times $S_1,\ldots, S_n$ are independent exponential random variables of parameters $q(Y_0),\ldots,q(Y_{n−1})$, respectively.

    Property 2 says that the next holding time is independent of previous holding times and jump times. The next state is sampled from the corresponding row of the jump matrix.

Construction I of a CTMC

  1. Let $(Y_n)_{n\leq0}$ be DTMC($\nu$,$R$)
  2. Let $T_1,T_2,\ldots$ be independent exponential R.V. of parameter 1, independent of $Y_n$.
  3. Set $S_n = \frac{T_n}{q(Y_{n−1})}$ to be the holding times, so that $S_n$ is a exponential R.V. with parameter $q(Y_{n-1})$.
  4. Let $J_n = S_1 + \ldots + S_n$ be the jump times.
  5. Let $X_t = \begin{cases}Y_n & \text{if } J_n \leq t < J_{n+1} \text{ for some } n, \ \infty & \text{otherwise}. \end{cases}$ be the right continuous process, which has $S_n$ as holding times and $Y_n$ as the jump chain.

Construction II of a CTMC

Use the transition rate matrix. Observations: Suppose $Y_n=i$, the holding time is $\exp(q_i)$, and the next state is sampled from the pmf ($r_{ij}:j\in X$).We can model this as follow:

Let $(w_j: j\in X, j\neq i)$ be iid exponential R.V. with rates $q_{ij}$. Let $S_{n+1}=\inf_{j\neq i} w_j$ with $Y_{n+1}$ being the minimizer. Then, $S_{n+1}\sim \exp(q_i)$ and $Y_{n+1}$ has pmf $(r_{ij}: j\in X)$ and $S_{n+1} \& Y_{n+1}$ are independent.

  1. Begin with an initial state $X_0=Y_0$ with distribution $\nu$.
  2. Let ($T^i_n:n\geq 1,j\in X$) be a collection of iid expo R.V. with parameter 1 . We will

  3. Then, inductively on $n=0,1,\ldots$, if $Y_n=i$ we set

    • $S_{n+1}^j = \frac{T^i_{n+1}}{q_{ij}}, \text{for} j\neq i$,
      For each possible next state j.
    • $S_{n+1} = \inf_{j\neq i} S_{n+1}^j$,
      Choose the shortest one.
    • $Y_{n+1} = \begin{cases}j & \text{if } S_{n+1}^j=S_{n+1}<\infty, \ i & \text{if} S_{n+1}=\infty\end{cases}$
      Determine the next state.

By this construction, $q_{ij}$ is called the rate of going from $i$ to $j$.
