ISyE 6761 Lec 18 CTMC Definition

Posted on 2024-11-26 In Course Notes , ISyE 6761

Jump Matrix

Def
: Given a transition-rate matrix $Q$ on an alomost countable set $I$, the jump matrix $R=(r_{ij}: i,j\in I)$ of $Q$ is defined as follows. For any $i\neq j, i\in X$,

  1. $r_{ij}=0, r_{ii}=1$ if $q_i=0$.
  2. $r_{ij}=\frac{q_{ij}}{q_i}, r_{ii}=0$ if $q_i\neq0$.

    For each row with $q_i\neq0$, the diagonal is set to 0 and normalize the other entries. When $q_i=0$, the diagonal is set to 1.
    There is no self-loops, unless it’s on absorbing state

Def
: A minimal right continuous process $(X_t)_{t\geq0}$ on state space X is a continuous time markov chain with initial distribution $\nu$ and generator matrix $Q$ if

  1. its jump chain $(Y_n)_{n\geq0}$ is DTMC($\nu$, $R$), where $R$ is the jump matrix of $Q$.
  2. for each $n\geq1$, conditional on $Y_0,\ldots Y_{n-1}$, its holding times $S_1,\ldots, S_n$ are independent exponential random variables of parameters $q(Y_0),\ldots,q(Y_{n−1})$, respectively.

    Property 2 says that the next holding time is independent of previous holding times and jump times. The next state is sampled from the corresponding row of the jump matrix.

Construction I of a CTMC

  1. Let $(Y_n)_{n\leq0}$ be DTMC($\nu$,$R$)
  2. Let $T_1,T_2,\ldots$ be independent exponential R.V. of parameter 1, independent of $Y_n$.
  3. Set $S_n = \frac{T_n}{q(Y_{n−1})}$ to be the holding times, so that $S_n$ is a exponential R.V. with parameter $q(Y_{n-1})$.
  4. Let $J_n = S_1 + \ldots + S_n$ be the jump times.
  5. Let $X_t = \begin{cases}Y_n & \text{if } J_n \leq t < J_{n+1} \text{ for some } n, \ \infty & \text{otherwise}. \end{cases}$ be the right continuous process, which has $S_n$ as holding times and $Y_n$ as the jump chain.

Construction II of a CTMC

Use the transition rate matrix. Observations: Suppose $Y_n=i$, the holding time is $\exp(q_i)$, and the next state is sampled from the pmf ($r_{ij}:j\in X$).We can model this as follow:

Let $(w_j: j\in X, j\neq i)$ be iid exponential R.V. with rates $q_{ij}$. Let $S_{n+1}=\inf_{j\neq i} w_j$ with $Y_{n+1}$ being the minimizer. Then, $S_{n+1}\sim \exp(q_i)$ and $Y_{n+1}$ has pmf $(r_{ij}: j\in X)$ and $S_{n+1} \& Y_{n+1}$ are independent.

  1. Begin with an initial state $X_0=Y_0$ with distribution $\nu$.
  2. Let ($T^i_n:n\geq 1,j\in X$) be a collection of iid expo R.V. with parameter 1 . We will

  3. Then, inductively on $n=0,1,\ldots$, if $Y_n=i$ we set

    • $S_{n+1}^j = \frac{T^i_{n+1}}{q_{ij}}, \text{for} j\neq i$,
      For each possible next state j.
    • $S_{n+1} = \inf_{j\neq i} S_{n+1}^j$,
      Choose the shortest one.
    • $Y_{n+1} = \begin{cases}j & \text{if } S_{n+1}^j=S_{n+1}<\infty, \ i & \text{if} S_{n+1}=\infty\end{cases}$
      Determine the next state.

By this construction, $q_{ij}$ is called the rate of going from $i$ to $j$.

ISyE 6761 Lec 17 Continuous-time Markov Chain

Posted on 2024-11-22 Edited on 2024-11-26 In Course Notes , ISyE 6761

$Q$-matrices and their exponentials

The time of CTMCs is within $[0,\infty]$, so we should interpolate between the dicrete times $0, 1, 2,\ldots$
A natural way to interpolate the discrete sequence is by funtion ($e^{tq}: t\geq 0$), since $p^t=(e^{\log p})^t=e^{tq}$, where $q=\log p$.

For any matrix $Q$, the series $\displaystyle \sum_{k=0}^{\infty} \frac{Q^k}{k!}$ converges componentwise and we denote the limit by $e^Q$.

If two Q-matrices commute ($Q_1Q_2=Q_2Q_1$), then $e^{Q_1 + Q_2} = e^{Q_1} e^{Q_2}$.

Suppose we can find a matrix Q with $e^Q=P$, then $e^{nQ}=P^n$, which fills in the gap.

Theorem

: Let $Q$ be a matrix on a finite set I. Set $P(t) = e^{tQ}$. Then $(P (t) : t \geq 0)$ has the following properties:

  1. $P (s + t) = P (s)P (t)$ for all s, t (semigroup property );

  2. $(P (t) : t \geq 0)$ is the unique solution to the forward equation
    $\frac{d}{dt} P (t) = P (t)Q, P (0) = I$;

  3. $(P (t) : t \geq 0)$ is the unique solution to the backward equation
    $\frac{d}{dt} P (t) = QP (t), P (0) = I$;

  4. for $k = 0,1,2,\ldots$, we have
    $(\frac{d}{dt})^k|_{t=0} P(t) = Q^k$

The above result is for arbitrary matrix $Q$. Next define a stochastic matrix.

$Q$-matrix

: Let $I$ be a countable set. A $Q$-$matrix$ $Q=(q_{ij}:i,j\in I)$ on $I$ is called a transition-rate matrix if it satisfies the following:

  1. $0\leq-q_{ii}\leq\infty, \forall i$
  2. $q_{ij}\geq 0, \forall i\neq j$

  3. $\displaystyle \sum_{j\in I}q_{ij}=0, \forall i$

    transition-rate matrix = $Q$-$matrix$ = intensity matrix = generator

    Define $q_i=\sum_{j\neq i} q_ij<\infty$, then $q_{ii}=-q_i$

    Each off-diagonal entry $q_{ij}$ gives the value we attach to the $(i, j)$ arrow on the diagram, which we shall interpret later as the rate of going from i to j. We shall later interpret qi as the rate of leaving i.

Theorem

: A matrix $Q$ on a finite set $\mathbb{X}$ is a transition-rate matrix if and only if $P(t)=e^{tQ}$ is a stochastic matrix for all $t\geq 0$.

If $P$ is a stochastic matrix of the form $e^Q$ for some $Q$-matrix, we can interpolate it between integers times. Fix a positive integer $m$, define a DTMC($\lambda$, $e^{Q/m}$) for $X_n^m$, and define $X_n^m = X_{n/m}$. Then $(X_n:n=0,1,\ldots)$ is a DTMC($\lambda$, $e^Q=P$).

Thus, by picking $m$ large enough, we can define a continuous time process $(X_t)t\geq 0$ has this property.

A continuous-time Markov chain $(Xt)_{t\geq0)}$ with $Q$-matrix satisfies

  • $P(X_{t_{n+1}})=i_{n+1}|X_{t_0} =i_0,\ldots,X_{t_n}=i_n)=P_{i_ni_{n+1}}(t_{n+1} −t_n)$

for all $n = 0,1,2,\ldots$, all times $0 \leq t_0 \leq \ldots \leq t_{n+1}$ and all states $i_0,\ldots ,i_{n+1}$, where $p_{ij}(t)$ is the $(i,j)$ entry in $e^{tQ}$. In particular, the transition probability from $i$ to $j$ in time $t$ is given by $P_i(X_t=j):=P(X_t=j|X_0=i)=p_{ij}(t)$.

Continuous-time Random Process and Explosion

For a right continuous stochastic process taking values in a countable set, every path must remain constant for a while in each new state.

Below are the three possibilities for the types of path:

  1. Infinitely many jumps , but finite many in any interval $[0,t]$.

  2. Finite many jumps and stuck in some state forever.

  3. Infinitely many jumps in a finite interval.

    $\zeta$ is the explosion time and the process restarts again after explosion. It may explode again or it may not.

The $J_i$ are jump times, and $S_i$ are holding times. If $J_{n+1}=\infty$, we define $X_\infty=X_{J_n}$.

Explostion time

The (first) explostion time $\zeta$ is defined by $\zeta=\sup_n J_n=\sum_{n+1}^\infty S_n$.

Jump Process

The discrete-time process $(Y_n)n\geq0$ given by $Y_n=X_{j_n}$ is called the jump process of $(X_t)t\geq0$, or jump chain if it’s a DTMC.

Minimal Process

We shall not consider what happens to a process after explosion. So we define a minimal process such that $X_t=\infty$ for $t\geq \zeta$. Here we adjoin a new state $\infty$ to $\mathbb{X}$.

“minimal” refers to the interval of time when the process is active.

A minimal process may be reconstructed from its holding times and jump process. Thus by specifying the joint distribution of $S_1, S_2, …$ and $(Y_n)n\geq0$, we can specify the probabilistic behaviour of $(X_t)t\geq0$ in terms of countable collection of R.V.

For example, $P(X_t=i)=\sum_{n=0}^\infty P(Y_n=i and J_n\leq t\leq J_{n+1})$ and
$P(X_t=i \text{ for some } t \in [0,\infty))=P(Y_n=i \text{ for some } n\geq 0)$

Birth Process

A birth process is a generalization of a Poisson process in which the parameter $\lambda$ is allowed to depend on the current state of the process.

A minimal right-continuous process $(X_t)_{t\geq 0}$ with values in ${0,1,2,\ldots}\cup{\infty}$ is a birth process of rates $0 \leq q_j < \infty$ if conditional on $X_0 = i$, its holding times are independent exponential random variables of rate $q_j$ and jump chain is given by $Y_n=i+n$. Explode or not

Theorem
: Let $(X_t)_{t\geq 0}$ be a birth process of rates ($q_j:j\geq0$), starting from 0.

  1. If $\sum_{j=0}^\infty \frac{1}{q_j}<\infty$, then $P(\zeta<\infty)=1$. i.e. explodes in finite time with prob 1.
  2. If $\sum_{j=0}^\infty \frac{1}{q_j}=\infty$, then $P(\zeta=\infty)=1$.
    i.e. does not explodes in finite time with prob 1.

If $\lambda_i = i^2$, it explodes. If $\lambda_i=i$, it doesn’t.
